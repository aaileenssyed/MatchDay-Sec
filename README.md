# MatchdaySec — Operation Kickoff

Standalone terminal-style SOC/NOC learning mission hosted separately from the SideQuest Cup hub.

## What it is

MatchdaySec is a guided terminal-style incident response lab. The user enters an analyst name, receives a mission brief from Operation Command, uses slash commands like `/logs`, `/status`, `/hint`, and `/playbook`, then makes decisions to keep matchday services online.

## Tech

- Next.js
- React
- TypeScript
- Tailwind CSS
- Static/session-only data
- No login
- No database

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Safety

This is an educational project using simulated data only. It does not scan, monitor, test, or represent real event infrastructure.
